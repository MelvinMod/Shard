# Welcome to Shard

## What is Shard?

Shard is a programming language. You use it to tell the computer what to do.

## What Can You Make?

- **Games** - Fun programs with pictures and sounds
- **Apps** - Tools you use every day
- **Websites** - Pages people visit on the internet
- **AI** - Smart programs that learn
- **Scripts** - Automatic tasks for your computer

## How to Start

### Step 1: Install Shard

Open your computer's terminal and type:

```
git clone https://github.com/MelvinMod/Shard.git
cd Shard
cargo build --release
```

### Step 2: Write Your First Program

Create a file called `hello.shard`:

```shard
say "Hello!"
```

### Step 3: Run It

```
shard run hello.shard
```

You will see:
```
Hello!
```
